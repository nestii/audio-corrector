# audio-corrector

Frontend: React with Vite  
Backend: Express

![App screenshot](assets/app-screenshot.png)

## Run

```bash
# Backend (port 3001)
cd backend && npm run dev

# Frontend (port 5173)
cd frontend && npm run dev
```

## Environment

Create `backend/.env` with:

```
OPENAI_API_KEY=your-key
```
